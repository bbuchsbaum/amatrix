# Construct an adlgCMatrix from a sparse or dense logical matrix

Construct an adlgCMatrix from a sparse or dense logical matrix

## Usage

``` r
new_adlgCMatrix(
  x,
  preferred_backend = .amatrix_default_preferred_backend(policy, precision),
  policy = amatrix_default_policy(),
  precision = amatrix_default_precision()
)
```
